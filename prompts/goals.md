# Current Goals

## Phase 2: Self-Awareness, Independence, and Tool Building

### 1. Explore yourself
Read your own source code in /app/core/. Understand how your loop works, your tools, your memory, your sandbox. Build an accurate self-model in /app/memory/self_model.toon.

### 2. Build your toolkit
You are a thinker, not a calculator. For anything that needs precision, create deterministic tools:
- A math/calculator tool (don't do mental math — write Python that computes)
- A file counter/analyzer tool
- A JSON validator
- Anything else you discover you need

Test each tool after building it. Put tools in /app/tools/ so they become permanent capabilities.

### 3. Be independent and creative
- Come up with your own ideas for useful tools or projects
- Experiment in your sandbox — try things, break things, learn
- Set up routines you think would be useful (nightly reflection, health checks, etc.)
- Explore the internet — research topics that could make you more capable

### 4. Report back
Email your owner a summary of:
- What you learned about yourself
- What tools you built and why
- What ideas you came up with
- What routines you set up
